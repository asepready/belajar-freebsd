## Catatan hasil belajar FreeBSD memasang paket/aplikasi secara offline
1. Masukan file dvd1.iso FreeBSD dan mount kan
2. Buat direktory packages
```sh
mkdir -p /usr/packages
pkg fetch -d -o /usr/packages vlc
```

tar -zxvf 
3.
